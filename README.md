# Project Launcher

一个高性能的本地项目启动器，帮助你快速搜索并打开本地代码项目。

## 核心特性

- 🚀 **高性能扫描** - 并行扫描，10000+ 项目秒级完成
- 🔍 **智能搜索** - 模糊匹配，实时过滤
- ⚡ **全局快捷键** - 随时唤醒，即搜即用
- 🎯 **智能排序** - 基于使用频率自动排序
- 🔧 **多启动器** - 支持配置多种编辑器/IDE
- 🌙 **系统集成** - 托盘常驻、开机自启
- 💻 **跨平台** - Windows / macOS

## 安装

从 [Releases](../../releases) 页面下载对应平台的安装包：

- **Windows**: `.msi` 或 `.exe`
- **macOS**: `.dmg`

## 使用方法

### 快速开始

1. 安装并启动应用，图标会出现在系统托盘
2. 在设置中配置启动器并设置快捷键
3. 按快捷键唤醒搜索框，输入项目名称进行搜索
4. 按 `Enter` 或点击项目使用对应启动器打开

### 搜索窗口

| 操作     | 快捷键        |
| -------- | ------------- |
| 上下选择 | `↑` / `↓`     |
| 打开项目 | `Enter`       |
| 快捷菜单 | `→` / 右键    |
| 关闭窗口 | `Esc`         |

### 配置设置

右键点击托盘图标，选择「设置」进入配置界面。

#### 工作区管理

添加你的代码目录，应用会自动扫描其中的项目（支持 Git / SVN / Mercurial 仓库识别）。

#### 启动器管理

配置你常用的编辑器或 IDE：

- VS Code、Cursor、WebStorm 等主流编辑器
- 支持自定义启动命令
- 可设置默认启动器

#### 快捷键设置

自定义全局唤醒快捷键，支持冲突检测。

#### 其他设置

- 开机自启
- 项目扫描深度
- 外观主题

### 托盘菜单

- **刷新缓存** - 重新扫描所有工作区
- **设置** - 打开设置窗口
- **退出** - 关闭应用
