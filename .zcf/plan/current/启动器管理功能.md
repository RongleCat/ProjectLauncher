# 启动器管理功能

## 任务描述
实现完整的启动器管理功能，包括创建、编辑、删除启动器，支持应用选择、自定义命令、独立快捷键绑定。

## 功能需求
1. 创建启动器：选择应用程序，自动生成默认命令
2. 编辑启动器：修改名称、路径、命令、快捷键
3. 删除启动器
4. 快捷键绑定：每个启动器可独立绑定全局快捷键
5. 命令模式：支持 `{project}` 占位符
